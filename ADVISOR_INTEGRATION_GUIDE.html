1️⃣ Cài đặt dịch vụ Advisor

a) Cách 1: Sử dụng Docker Compose (Khuyến nghị)

# Khởi động tất cả services bao gồm advisor-service
cd /path/to/ktra1
docker-compose up -d

# Kiểm tra advisor-service đang chạy
docker-compose ps | grep advisor
    
# Logs
docker-compose logs -f advisor-service

b) Cách 2: Chạy trực tiếp (Development)

# Tạo virtual environment
cd advisor_service
python -m venv venv

# Activate
# Windows
venv\Scripts\activate
# Linux/Mac
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Set environment variables
cp .env.example .env
# Chỉnh sửa .env

# Migration
python manage.py migrate

# Tải KB sample
python manage.py load_kb --sample

# Chạy server
python manage.py runserver 0.0.0.0:8003
✓ Kiểm tra: Truy cập http://localhost:8003/admin và đăng nhập

2️⃣ Tích hợp Chatbot vào Website

Phương pháp A: Nhúng trực tiếp (Đơn giản)

Thêm đoạn code sau vào cuối phần <body> của tất cả trang HTML:

<!-- AI Advisor Chatbot -->
<script src="/ai-advisor.js"></script>
<link rel="stylesheet" href="/ai-advisor.css">

Phương pháp B: Sử dụng iframe (An toàn hơn)

<iframe 
    src="/ai-advisor-embed.html" 
    style="position: fixed; right: 20px; bottom: 20px; width: 430px; height: 600px; border: none; border-radius: 12px; z-index: 9999;"
></iframe>

Phương pháp C: Tích hợp vào component React (Nên dùng)

import React, { useEffect } from 'react';

export function AIAdvisor() {
  useEffect(() => {
    // Load script
    const script = document.createElement('script');
    script.src = '/ai-advisor.js';
    script.async = true;
    document.body.appendChild(script);
    
    // Load styles
    const link = document.createElement('link');
    link.rel = 'stylesheet';
    link.href = '/ai-advisor.css';
    document.head.appendChild(link);
  }, []);

  return <> {/* Styles will be injected */} </>;
}

Cách tích hợp vào trang index.html

<!DOCTYPE html>
<html>
<head>
    <title>E-Commerce</title>
</head>
<body>
    <!-- Nội dung website -->
    
    <!-- AI Advisor - Thêm dòng này ở cuối body -->
    <iframe src="/ai-advisor.html" 
            style="position: fixed; right: 20px; bottom: 20px; width: 430px; height: 600px; border: none; border-radius: 12px; z-index: 9999; box-shadow: 0 5px 40px rgba(0,0,0,0.16);">
    </iframe>
</body>
</html>

3️⃣ Cấu hình API

Cập nhật nginx.conf để proxy API

server {
    listen 80;
    server_name _;

    # ... existing config ...

    # API Gateway
    location /api/advisor/ {
        proxy_pass http://advisor-service:8003/api/;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
        proxy_set_header X-Forwarded-For $proxy_add_x_forwarded_for;
        proxy_set_header X-Forwarded-Proto $scheme;
    }

    # Frontend
    location / {
        root /usr/share/nginx/html;
        try_files $uri $uri/ /index.html;
    }
}

Cập nhật CORS trong advisor_service/settings.py

CORS_ALLOWED_ORIGINS = [
    "http://localhost:3000",
    "http://localhost:8000",
    "http://localhost:8100",  # Gateway
    "http://172.17.0.1:3000",
]

4️⃣ Tải Knowledge Base

Tải KB trong Docker

# Vào shell của advisor-service
docker-compose exec advisor-service bash

# Tải KB sample
python manage.py load_kb --sample

# Hoặc tải từ file JSON
python manage.py load_kb --file /path/to/kb.json

5️⃣ Test & Debug

Test API Chatbot

# Start chat
curl -X POST http://localhost:8003/api/advisor/sessions/start_chat/ \
  -H "Content-Type: application/json" \
  -d '{"conversation_type": "consultation"}'

# Send message
curl -X POST http://localhost:8003/api/advisor/sessions/send_message/ \
  -H "Content-Type: application/json" \
  -d '{
    "session_id": "YOUR_SESSION_ID",
    "message": "Hỏi về laptop Apple"
  }'
